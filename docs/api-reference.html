API Reference

Complete reference for MeroJs, all sub-clients, and utility functions

MeroJs (main class)

class MeroJs {
  auth: AuthApiClient;
  admin: AdminApiClient;
  rpc: RpcClient;
  sse: SseClient;
  ws: WsClient;

  constructor(config: MeroJsConfig);
}

interface MeroJsConfig {
  baseUrl: string;       // Node HTTP base URL
  timeoutMs?: number;    // Request timeout (default: no timeout)
  tokenStore?: TokenStore;// Custom token storage
  requestCredentials?: RequestCredentials;// fetch credentials mode
}

AuthApiClient — sdk.auth

login(params)
Authenticates with username + password. Stores tokens in TokenStore. Returns TokenData.
logout()
Revokes tokens and clears TokenStore.
refresh()
Exchanges the stored refresh token for a new access token. Called automatically before expiry.
initFromTokens(tokens)
Initialises auth state from pre-existing tokens (e.g. from URL hash SSO). Does not call the network.
isAuthenticated()
Returns true if a valid (non-expired) access token is available.

AdminApiClient — sdk.admin

getContexts()
List all contexts on the node. Returns Context[].
getContext(contextId)
Get a single context by ID with full detail.
createContext(params)
Create a new context, optionally installing an application at the same time.
getApplications()
List all installed applications.
installApplication(params)
Install an application into a context by package ID.
getIdentities()
List node identities.
generateClientKey(params)
Generate a client signing key for a context. Params: contextId, contextIdentity (camelCase in v2).

RpcClient — sdk.rpc

execute(params)
Execute a WASM application method. params: contextId, method, argsJson. Returns raw JSON response.
query(params)
Alias for execute — read-only operations. Same signature.
mutate(params)
Alias for execute — state-changing operations. Same signature.

Utility Functions

parseAuthCallback(hash)
Parses window.location.hash for SSO tokens. Returns { accessToken, refreshToken, nodeUrl, applicationId } or null.
buildAuthLoginUrl(options)
Constructs the URL for redirecting to the auth frontend. Used when triggering the full auth flow from an app.
HTTPError
Error class thrown on non-OK HTTP responses. Properties: status, statusText, body.
