Getting Started

Install, connect, and make your first API call in minutes

Installation

npm

npm install @calimero-network/mero-js

pnpm

pnpm add @calimero-network/mero-js

yarn

yarn add @calimero-network/mero-js

Requires Node.js 18+ or any modern browser. No polyfills needed.

Basic Example

1

Create an SDK instance

import { MeroJs } from '@calimero-network/mero-js';

const sdk = new MeroJs({
  baseUrl: 'https://your-node.example.com',
  timeoutMs: 10_000,
});
2

Authenticate

// Login with username and password
await sdk.auth.login({
  username: 'admin',
  password: 'your-password',
});

// Or initialise from existing tokens (e.g. from URL hash SSO):
import { parseAuthCallback } from '@calimero-network/mero-js';
const tokens = parseAuthCallback(window.location.hash);
if (tokens) await sdk.auth.initFromTokens(tokens);
3

List contexts

const { data: contexts } = await sdk.admin.getContexts();
console.log(contexts); // array of context objects
4

Execute an application method

const result = await sdk.rpc.execute({
  contextId: contexts[0].id,
  method: 'get_value',
  argsJson: JSON.stringify({ key: 'hello' }),
});
console.log(result);

SSO — Reading Tokens from URL Hash

When your app is opened from Calimero Desktop, tokens arrive in the URL hash. Use parseAuthCallback():

import { MeroJs, parseAuthCallback } from '@calimero-network/mero-js';

const auth = parseAuthCallback(window.location.hash);
if (auth) {
  // auth = { accessToken, refreshToken, nodeUrl, applicationId }
  const sdk = new MeroJs({ baseUrl: auth.nodeUrl });
  await sdk.auth.initFromTokens({
    accessToken: auth.accessToken,
    refreshToken: auth.refreshToken,
  });
  // Clear hash from URL (security best practice)
  history.replaceState(null, '', location.pathname);
}

Error Handling

import { HTTPError } from '@calimero-network/mero-js';

try {
  const result = await sdk.rpc.execute(...);
} catch (err) {
  if (err instanceof HTTPError) {
    console.error(`HTTP ${err.status}: ${err.statusText}`);
  } else if (err.name === 'AbortError') {
    console.log('Request cancelled or timed out');
  } else {
    throw err; // network error, rethrow
  }
}

See Error Model for full details on all error types and handling strategies.
