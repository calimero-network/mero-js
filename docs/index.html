SDK Architecture

Features

◆

Zero Dependencies

Uses only Web Standards: fetch, AbortController, URL. No axios, no node-fetch, no bloat.

 ◆

Smart Error Handling

Throws HTTPError on non-OK responses. Distinguishes user aborts from timeouts for intelligent retry logic.

 ◆

Real-time Events

Server-Sent Events (SSE) and WebSocket subscription clients for real-time node event streams.
◆

TypeScript First

Full TypeScript definitions for all API clients, request/response types, and configuration objects.

 ◆

Advanced Cancellation

Signal combination — merge multiple AbortSignals and timeout signals so requests clean up correctly.

 ◆

Universal Runtime

Works in the browser, Node.js 18+, Deno, Bun, and edge runtimes like Cloudflare Workers.
